# doSub()

> This function is used to subtract 2 numbers.

- @param {number} num1 First number
- @param {number} num2 Second number
- @return The subtraction of the two numbers

## Syntax

function(a, b) { ..code.. }

## Parameters

The function takes two parameters and returns the output.

## Example

### Code:

```js
const calculator = require("all-round-calculator");
console.log(calculator.doSub(5, 7));
console.log(calculator.doSub(25, 7));
```

### Output:

```bash
-2
18
```
